### Week 1 Objectives:
* Get to know the core concepts of Cloud Computing and the AWS Global Infrastructure.
* Securely set up an AWS account using Multi-Factor Authentication (MFA) and Identity and Access Management (IAM) for admin users.

### Week 1 Achievements:

* Understand the basic concepts of Cloud Computing and how AWS works, including core services like Compute, Storage, Networking, and Database.

* Successfully created an AWS account. Protected the account by setting up Multi-Factor Authentication (MFA) and configured admin groups and users for better security.

* Learned how to monitor and optimize costs on AWS. Successfully practiced creating different types of budgets, such as Usage, Reservation Instance (RI), and Savings Plans budgets.

* Know how to use the AWS Support system, understand different support packages, and know how to create and manage support requests when problems occur.

* Became familiar with the AWS Management Console, which allows for easy navigation, access, and management of cloud resources.
